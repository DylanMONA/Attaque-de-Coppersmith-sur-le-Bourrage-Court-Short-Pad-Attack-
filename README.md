#Attaque de Coppersmith sur le Bourrage Court (Short Pad Attack)

#Description

Ce projet présente une étude théorique et mathématique de l’attaque de Coppersmith appliquée à RSA avec bourrage court (Short Pad Attack), notamment lorsque l’exposant public est petit (par exemple e = 3).

L’objectif est de montrer comment un attaquant peut exploiter deux chiffrements du même message avec des bourrages aléatoires différents mais courts, afin de retrouver la différence de bourrage, puis le message original, en utilisant :

le théorème de Coppersmith,

l’algorithme LLL,

le lemme de Howgrave-Graham,

et le théorème de Franklin-Reiter.

Ce projet est principalement pédagogique et destiné à illustrer une faiblesse théorique de RSA mal utilisé
